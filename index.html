Guía de Git

Comandos básicos.

Inciar git en consola local

git init

Configurar usuario y email

git config --global user.username "Nombre"

git config --global user.email "Dirección email"

Añadir ficheros al registro de git

git add "nombreFichero.extension"

Comprobar estado del repositorio

git status

Registro de commits

git log
